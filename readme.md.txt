# Czerwone Łąki – strona www

Struktura:

- `index.html` – cała strona (Premium Forest Minimal, pole namiotowe, rowery, mapa, formularz rezerwacji)
- `logo-czerwone-laki.svg` – proste logo w stylu leśno-minimalistycznym

## Jak uruchomić lokalnie

1. Otwórz plik `index.html` w przeglądarce (dwuklik).
2. Formularz rezerwacji używa `mailto:` – otworzy klienta poczty użytkownika.

## Jak wrzucić na GitHub + Vercel

1. Utwórz repozytorium na GitHub.
2. Dodaj folder `czerwone-laki` z plikami:
   - `index.html`
   - `logo-czerwone-laki.svg`
   - `README.md`
3. Na Vercel:
   - Importuj repo z GitHuba.
   - Jako root projektu wybierz folder `czerwone-laki`.
   - Deploy.

Strona będzie dostępna pod adresem wygenerowanym przez Vercel (np. `https://czerwone-laki.vercel.app`).
